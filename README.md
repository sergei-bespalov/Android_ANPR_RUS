# Android ANPR

Приложение распознает автомобильные номера и выводит на экран в режиме реального времени.

Используются следующие библиотеки:

* [https://code.google.com/p/tesseract-android-tools/](https://code.google.com/p/tesseract-android-tools/)
* [http://opencv.org/](http://opencv.org/)

Из-за небольшого размера выборки для обучения каскадного классификатора стабильно работает только на тестовых примерах. Необходимо изменение алгоритма бинаризации, метод Оцу затирает числа если они не четкие.
